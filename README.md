# Toledo Utilidades

Coleção de ferramentas web úteis para o dia a dia, desenvolvida com HTML, CSS e JavaScript puro — sem frameworks ou dependências externas.

🔗 **[Acesse o site](https://mtoledo2.github.io/utilidades/)**

## Ferramentas disponíveis

### 🎲 Sorteador de Grupos
Adicione participantes, defina quantos grupos deseja e o sorteador distribui automaticamente de forma equilibrada. Ideal para dividir times, duplas ou qualquer agrupamento aleatório.

### 🧮 Calculadoras
Conjunto de calculadoras organizadas por categoria:

- **Saúde:** IMC, ingestão diária de água, Taxa Metabólica Basal (TMB)
- **Treino:** Repetição máxima (1RM), macronutrientes diários
- **Jogos:** eDPI, conversor de sensibilidade, FOV ideal, tempo de grinding
- **Geral:** Tempo entre datas

## Tecnologias

- HTML5
- CSS3 com variáveis CSS e design responsivo
- JavaScript vanilla (sem frameworks)
- Font Awesome 6 para ícones
- Google Fonts (Syne + DM Sans)
- GitHub Pages para deploy

## Destaques técnicos

- Tema claro/escuro com detecção automática do sistema operacional
- Menu lateral responsivo com comportamento adaptado para mobile e desktop
- Arquitetura modular com separação de responsabilidades entre HTML, CSS e JS
- Sem dependências de build — funciona diretamente no navegador
